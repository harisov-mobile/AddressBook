
Примечания:

- Просмотр сделан в отдельной активити, добавление/редактирование - в своей активити.

- используется FAB - floating action button.

- данные хранятся в SQLite.

- В RecyclerView применен разделитель, который через класс ItemDivider сделан.

- "Редактировать" и "Удалить" выведены в меню ContactFragment.

- для планшета "список/детализация" - сделан на одном экране, сделан вертикальный разделитель
(разделитель можно по разному делать)

- для телефона сделан ViewPager

- фото сохраняется в файл в папку приложения.

- при удалении контакта - диалоговый вопрос пользователю. Потом передача информации о том, что выбрал пользователь, между фрагментами.

- в списке отображается иконка фото контакта.

- для телефона заблокирован поворот (на телефоне в повернутом экране очень плохо смотрится/редактируется).

- сохранение фотографии во временный файл, если пользователь нажал кнопку сохранить - тогда в правильный UUID-файл.

- Picasso: https://square.github.io/picasso/

- При нажатии на иконку-фото - показ увеличенной фотографии (в Dialog)
